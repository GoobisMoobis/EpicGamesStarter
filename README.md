# EpicGamesStarter

Due to the "deprecation" of the original project, and the fact antiviruses would have a heart attack whenever someone tried to install it, I've re-made EpicGamesStarter so that Windows takes it easier.

I made two versions of this program. I personally use the .ps1 but the .bat is reccomended for everyday use.

| Filetype | Function | ‎File size‎ | Notes |
| :------- | :------- | :--- | :---- |
| .bat     | Best for everyday use. Starts Among Us regularly and gets the authentication from the command line arguments | 2 KB | Faster |
| .ps1     | Uses a cached token to start the game, requires login. If an error occurs it will be easier to pinpoint | 6 KB | Slower than the .bat, faster than it after first use. |
